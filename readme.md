# 春节倒计时

春节倒计时 是一个用于显示距离春节还有多久的网站。

## 特性

- 提供与春节相关的节假日信息。
- 简单易用，轻量级。

## 安装

通过 npm 安装：

```bash
npm install cny
```

或者通过 yarn 安装：

```bash
yarn add cny
```
